Shell Interpreter in C

This project involved designing and developing a shell interpreter in C demonstrating proficiency in low-level programming and system call interaction. As part of the project I implemented core shell functionalities, including command parsing, thread-safe process execution, pipelining, and I/O redirection, while adhering to Unix-like shell standards. Through development and design of this project I gained valuable insights into systems programming and the Unix environment, furthering proficiency of software development in resource-constrained systems.

Rock Paper Scissors Algorithm Analysis

For this project, two of my peers and I were asked to build a program that could place two user-created rock paper scissors algorithms against each other, analysing the results of potentially thousands of matches to determine the victorious algorithm. Included are each of our distinct algorithms that we designed from the ground up, teaching us important design considerations.

C++ Grid-Based City Simulation

This project required two of my peers and I to design and develop a console-based city simulation in C++ that showcases population growth and pollution over a period of time set by the user. Work on the project encouraged us to create effective, readable design documents and work with unfamiliar 2-dimensional, dynamic arrays in order to ensure the project reached its deadline.
